Report Editor Instructions

Overview

This editor is designed to act as an easy-to-use interface for creating or modifying a SARE report.

Submission Options

  • Save - Save your current changes
  • Save and Preview - Save your changes and preview them in the report viewer
  • Cancel - Return to the project overview page without saving current changes

Upload Files

Maximum upload size is 30 Mb for documents and 140 Mb for media files. Tips for reducing image and file size.

Allowable File Types: Documents(.doc,.docx,.pdf,.rtf) Figures (ppsx,.ppt,.pptx) Tables (.xls,.xlsx), Images (.gif,.jpeg,.jpg,.png) Video (.asf,.asx,.avi,.mov,.mp4,.mpeg,.mpg,.qt) and audio (.au,.mp3)

Assign Project Code

Use this feature to assign a new project code to a pi or co-pi that has not yet claimed his or her project. Persons who claim a project will be listed as a coordinator in the general information section. Do NOT assign new project codes to pis that have already claimed their project- they can access the project by logging into their MySARE account and clicking My Projects. A code can be used by more than one project coordinator to claim a project, but issuing a new project code will invalidate previously issued codes. When providing multiple coordinators with access, issue just one code, separating their email addresses with a comma.

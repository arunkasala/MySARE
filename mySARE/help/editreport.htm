Report Editor Instructions

Overview

This editor is designed to act as an easy-to-use interface for creating or modifying a SARE report. Based on the current project details and report type, the editor will automatically create an outline suitable for your specific report. The editor provides multiple controls for modifying the order and display type of subsections within these headings.

Creating and Deleting Subsections

To the right of each subsection, you will see two control buttons. Their functions are as follows.

  • Insert Subsection Below - Create a new subsection directly below the current one
  • Delete Subsection - Remove the current subsection from the report

Subsection Types

The subsection type dropdown box controls how the section's text will be displayed when the report is converted to a printable format. Each text section can only support one format, so you may need to create additional text blocks to enter separate lists, paragraphs, etc. The available display modes and examples of their use are as follows.

  • Paragraph
  • Bulleted List
  • Numbered List
  • Quote Block
  • Unindented Paragraph

Submission Options
  • Save Without Submitting - Save your changes without submitting the report for approval
  • Save and Preview - Save your changes and preview them in the report viewer
  • Submit - Save and submit your proposal for approval
  • Cancel - Return to the project overview page without saving current changes

Upload Files

Maximum upload size is 30 Mb for documents and 140 Mb for media files. Tips for reducing image and file size.

Allowable File Types: Documents(.doc,.docx,.pdf,.rtf) Figures (ppsx,.ppt,.pptx) Tables (.xls,.xlsx), Images (.gif,.jpeg,.jpg,.png) Video (.asf,.asx,.avi,.mov,.mp4,.mpeg,.mpg,.qt) and audio (.au,.mp3)
